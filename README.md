  - pg서비스에서 결제건의 카드/간편/휴대폰/가상계좌 결제유형에 따라 해당pg의 결제서비스를 호출. <br>
  - 각pg사 연동코드는 전문기반으로 프로세스를 추상화하여 관리.
- **gRPC 내부 통신 최적화**
